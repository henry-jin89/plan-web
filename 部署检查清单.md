## 📤 第三步：上传文件到 GitHub

### 🖱️ 方法一：Web界面上传（推荐）

1. **进入仓库主页：**
   ```
   https://github.com/henry-jin89/plan-web
   ```

2. **开始上传：**
   - 点击 "uploading an existing file" 链接
   - 或者点击 "Add file" → "Upload files"

3. **选择文件：**
   - 将 `plan-web-main` 文件夹中的**所有文件**拖拽到上传区域
   - 或点击 "choose your files" 选择文件
   - **注意：** 要选择文件夹内的所有文件，不是文件夹本身

4. **确认上传：**
   - 页面底部填写提交信息：`初始部署：智能计划管理器`
   - 点击 "Commit changes"

### 💻 方法二：Git命令行（高级用户）

```bash

git clone https://github.com/henry-jin89/plan-web.git
cd plan-web

# 复制文件（将plan-web-main下的所有文件复制到当前目录）
# Windows: copy plan-web-main\* .
# Mac/Linux: cp plan-web-main/* .

# 添加文件
git add .

# 提交
git commit -m "初始部署：智能计划管理器"

# 推送
git push origin main
```

## 🌐 第四步：启用 GitHub Pages

### ⚙️ 配置步骤：

1. **进入仓库设置：**
   - 在仓库页面点击 "Settings" 标签

2. **找到 Pages 设置：**
   - 在左侧菜单向下滚动
   - 点击 "Pages"

3. **配置部署源：**
   ```
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
   ```

4. **保存设置：**
   - 点击 "Save" 按钮

5. **等待部署：**
   - 页面顶部会显示：
   ```
   ✅ Your site is published at https://henry-jin89.github.io/plan-web/
   ```
   - 初次部署需要 1-5 分钟

### 🔍 验证部署：
- 访问：`https://henry-jin89.github.io/plan-web/`
- 应该能看到计划管理器主页
- 检查移动端适配是否正常

## 🔄 第五步：配置数据同步

### 📊 创建数据存储仓库：

1. **创建私有仓库：**
   - 再次点击 GitHub 右上角 "+"
   - 选择 "New repository"
   - 填写信息：
     ```
     Repository name: plan-data
     Description: 计划管理器数据存储（私有）
     Private: ✅ 选择 Private（私有）
     Add a README file: ✅ 勾选
     ```
   - 点击 "Create repository"

### 🔑 生成访问令牌：

1. **访问令牌设置页面：**
   ```
   https://github.com/settings/tokens
   ```

2. **生成新令牌：**
   - 点击 "Generate new token"
   - 选择 "Generate new token (classic)"

3. **配置令牌：**
   ```
   Note: 计划管理器数据同步
   Expiration: 90 days (或 No expiration)
   Scopes: ✅ repo (Full control of private repositories)
   ```

4. **生成并保存：**
   - 点击 "Generate token"
   - **重要：** 立即复制并保存token（只显示一次）
   - 格式类似：`ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### ⚙️ 配置同步设置：

1. **访问同步设置页面：**
   ```
   https://henry-jin89.github.io/plan-web/sync-settings.html
   ```

2. **选择 GitHub 同步：**
   - 点击 "GitHub 同步" 卡片

3. **填写配置信息：**
   ```
   Personal Access Token: ghp_xxxxx...（刚才生成的token）
   仓库所有者: henry-jin89
   仓库名称: plan-data
   分支名称: main
   ```

4. **测试连接：**
   - 点击 "测试连接" 按钮
   - 等待显示 "✅ 连接测试成功！"

5. **启用同步：**
   - 点击 "启用同步" 按钮
   - 系统会自动执行初始同步
