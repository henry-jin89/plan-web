# 🏠 联想个人云零配置同步 - 超简单设置

## ✨ 为什么选择联想个人云？

相比GitHub同步，联想个人云有以下优势：

- ✅ **更简单** - 只需要用户名密码，无需Token
- ✅ **更快速** - 本地网络，同步速度更快
- ✅ **更私密** - 数据完全在您家中，无需担心隐私
- ✅ **更稳定** - 不受互联网限制，局域网即可使用
- ✅ **自动发现** - 系统可以自动检测设备IP地址

## 🚀 超简单设置（仅需5分钟）

### 第一步：联想个人云设备准备

1. **启用WebDAV服务**
   - 打开联想个人云管理界面（通常是 http://设备IP）
   - 进入"应用中心"或"服务管理"
   - 找到并启用"WebDAV"服务
   - 确认端口设置为5005（默认）

2. **创建专用用户**
   - 进入"用户管理"
   - 点击"添加用户"
   - 用户名：`planSync`（推荐）
   - 设置密码（请记住）
   - 分配读写权限

### 第二步：获取设备IP地址

**方法一：从管理界面查看**
- 在联想云管理界面查看"系统信息"
- 找到网络配置中的IP地址

**方法二：自动检测（推荐）**
- 系统会自动尝试常见IP地址
- 192.168.1.100, 192.168.0.100 等

**方法三：从路由器查看**
- 登录路由器管理界面
- 查看已连接设备列表

### 第三步：配置同步服务

**方法一：使用配置文件（推荐）**

1. 复制配置文件：
   ```bash
   cp lenovo-config.js.example lenovo-config.js
   ```

2. 编辑 `lenovo-config.js`：
   ```javascript
   window.LENOVO_CLOUD_CONFIG = {
       enabled: true,
       url: 'http://192.168.1.100:5005',  // 您的设备IP
       username: 'planSync',              // 您创建的用户名
       password: '您的密码',               // 用户密码
       timeout: 30000,
       autoDetect: true
   };
   ```

**方法二：直接在代码中配置**

编辑 `lenovo-cloud-sync.js` 文件：
```javascript
this.defaultConfig = {
    enabled: true,
    url: 'http://192.168.1.100:5005',
    username: 'planSync',
    password: '您的密码',
    timeout: 30000,
    autoDetect: true
};
```

### 第四步：测试验证

1. **打开任意页面**（如日计划、习惯跟踪）
2. **查看浏览器控制台**（F12 → Console）
3. **寻找成功信息**：

## 🔍 故障排除

### Q: 提示"设备未找到"？
**A:** 检查以下几点：
- 联想个人云设备是否开机
- IP地址是否正确
- WebDAV服务是否启用
- 端口5005是否开放

### Q: 提示"认证失败"？
**A:** 检查：
- 用户名是否正确
- 密码是否正确
- 用户是否有读写权限

### Q: 提示"连接超时"？
**A:** 可能原因：
- 网络连接问题
- 设备响应慢
- 防火墙阻止连接

### Q: 如何查看设备IP？
**A:** 多种方法：
- 联想云管理界面 → 系统信息
- 路由器管理界面 → 设备列表
- 让系统自动检测（推荐）
