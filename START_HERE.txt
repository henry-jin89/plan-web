📋 部署步骤概览（手动部署）：

  第1步: 推送代码到 GitHub
         git add . && git commit -m "部署准备" && git push

  第2步: 在 Render.com 创建 Web Service
         访问: https://dashboard.render.com
         配置: Root Directory = websocket-server

  第3步: 配置前端连接
         运行: node add-websocket-config.js
         编辑: websocket-config.js

  第4步: 推送更新
         git add . && git commit -m "配置完成" && git push

  第5步: 测试部署
         运行: ./check-deployment.sh
